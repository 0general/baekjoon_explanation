# 코딩 테스트 대비 스터디

코딩 테스트를 준비하는 스터디를 위한 저장소입니다.
이 스터디는 그 주에 정해진 문제를 풀고 **매주 화요일** 설명하는 방식으로 이루어집니다.

그림과 설명을 준비하며 푼 문제의 유형에 대해 더 깊이 탐구하고
코드에 대한 정확한 이해와 더 나은 풀이를 끌어냅니다.

> ## 유형별 문제 풀이
> > + DP
> > + DFS/BFS
> > + 구현
> > + 그리디
